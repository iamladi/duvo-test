# Demo Data

Sample files for demonstrating the MCP filesystem connection in duvo-test.

## Contents

- **sales.csv** — Product sales data with dates, quantities, and totals
- **notes.md** — Project meeting notes and goals
- **config.json** — Application configuration settings

## Usage

These files are used as the default data source when enabling the filesystem connection in the duvo-test UI. The agent can read, list, and search these files via MCP tools.
